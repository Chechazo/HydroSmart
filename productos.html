SmartHeater

El calentador inteligente que revoluciona la forma en que utilizas el agua caliente en tu hogar. Eficiencia, tecnología y sustentabilidad en un solo producto.

¿Qué es SmartHeater?

SmartHeater es nuestro producto estrella: un boiler inteligente diseñado para calentar agua de manera eficiente, segura y automatizada. A diferencia de los sistemas tradicionales, este dispositivo solo funciona cuando realmente se necesita, reduciendo el consumo de energía y evitando el desperdicio de recursos.

Integra tecnología IoT que permite su control remoto mediante una aplicación móvil, además de monitorear en tiempo real variables como temperatura, consumo eléctrico y estado del sistema.

¿Cómo funciona?

El sistema utiliza un microcontrolador ESP32 conectado a sensores de temperatura y nivel. Cuando el usuario activa el calentador desde su celular, el sistema enciende automáticamente la resistencia eléctrica y comienza el proceso de calentamiento.

Una vez que el agua alcanza la temperatura deseada (entre 65 y 70 °C), el sistema se apaga de forma automática para evitar sobrecalentamiento y ahorrar energía.

Toda la información se sincroniza en la nube, permitiendo monitoreo y control desde cualquier lugar.

Características Principales

Encendido y apagado remoto desde una aplicación móvil en tiempo real.

Sistema de apagado automático ante fallas o sobrecalentamiento.

Base de datos en la nube para sincronización con la app móvil.
